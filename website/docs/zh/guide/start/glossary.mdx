## Bundleless

Bundleless 是指对每个源文件单独进行编译构建，但是并不将它们打包在一起。每一个产物文件都可以找到与之相对应的源码文件。Bundleless 构建的过程，也可以理解为仅对源文件进行代码转换的过程。
